
# Engineering Probability and Statistics Projects

This repository contains a collection of 4 practical assignments focused on implementing fundamental statistical concepts, probability distributions, and introductory Machine Learning models using Python.

**Author:** Shayan Maleki

---

## 📋 Projects Overview

### 1. NumPy & Naive Bayes Classifier (EPS CA0)
**Focus:** Scientific Computing & Machine Learning (Classification)
This project serves as an introduction to scientific computing and applies probability theory to a real-world text classification problem.
- **Key Tasks:** Implementing core array operations using **NumPy** and simulating the "Majority Voting" concept.
- **Machine Learning Component:** Built a **Naive Bayes Classifier** from scratch for **Spam Detection**. This involved:
  - Text preprocessing and cleaning.
  - Creating a **Bag-of-Words (BoW)** model to extract features.
  - Splitting data into Training/Testing sets and evaluating the model using accuracy metrics.

### 2. Statistical Distributions (EPS CA1)
**Focus:** Probability Theory & Simulation
A project dedicated to simulating and analyzing discrete probability distributions.
- **Key Tasks:** Simulating the **Hypergeometric** and **Binomial** distributions. The project involved generating random samples, plotting Probability Mass Functions (PMF), and comparing empirical simulation results with theoretical means and variances.

### 3. Bayesian Estimation & Correlation (EPS CA2)
**Focus:** Advanced Probability & Estimation
This assignment explores relationships between random variables and inference techniques.
- **Key Tasks:** Working with **Joint Distributions**, calculating the **Correlation Coefficient** to analyze linear relationships, and applying **Bayesian Estimation** methods to infer unknown parameters from data.

### 4. Central Limit Theorem & Linear Regression (EPS CA3)
**Focus:** Statistical Theorems & Machine Learning (Regression)
This project covers theoretical foundations and continuous predictive modeling.
- **Key Tasks:** Demonstrating the **Central Limit Theorem (CLT)** through simulation and performing Parameter Estimation.
- **Machine Learning Component:** Implemented **Linear Regression** to predict continuous target variables. This included calculating the **Mean Squared Error (MSE)** to evaluate the regression model's performance and accuracy.
